### About!

I'm Ryuichiro Hataya, a PhD student at UTokyo (supervisor: [Hideki Nakayama](http://www.nlab.ci.i.u-tokyo.ac.jp/~nakayama/index.html)) and RA at RIISE@UTokyo.
